[heading][color=#8ECFFB]Description[/color][/heading]
Wyrdlight ReShade is a performance-friendly dark fantasy preset that pulls Skyrim back toward its vanilla cold northern palette.

That's the main appeal: it makes Skyrim look like Skyrim. It leans into a darker, cooler image with a blue-purple tint, measured vibrance, and adaptive ambient light around brighter areas.

The preset focuses on tone, colour, exposure, black point, ambient light, and subtle film response. The result is crisp, moody, readable Skyrim with a darker fantasy edge while still feeling true to the original game. Fully equipped for the slaying of dragons.

[heading][color=#8ECFFB]Features[/color][/heading]
[list]
[*]Performance-friendly dark fantasy ReShade preset.[/*]
[*]Easy installation tool that downloads ReShade and configures paths, shortcut keys, and screenshots for you.[/*]
[*]Negligible performance impact with Ambient Light turned off; Ambient Light costs about 2 FPS when enabled.[/*]
[*]Restores a colder vanilla-Skyrim colour palette.[/*]
[*]Darkens and cools the image while keeping it readable.[/*]
[*]Adds measured vibrance and adaptive ambient light.[/*]
[*]Uses subtle filmic contrast, levels, and tone mapping.[/*]

[heading][color=#8ECFFB]Installation[/color][/heading]
[b]Recommended: use the setup tool.[/b] It downloads ReShade, configures paths, sets shortcut keys, and writes the screenshot folder for you.

[list=1]
[*]Install [b]Wyrdlight ReShade - Northern Skies.zip[/b] with Mod Organizer 2, or extract it manually.[/*]
[*]Open the installed mod in Explorer and run [b]root\Wyrdlight ReShade Setup.exe[/b].[/*]
[*]Accept the detected [b]Stock Game[/b] or Skyrim folder. If the tool cannot detect it, browse to the folder containing [b]SkyrimSE.exe[/b].[/*]
[*]For MO2 Root Builder, choose to configure the installed mod folder for Root Builder.[/*]
[*]Choose the latest full add-on ReShade build, shortcut keys, and screenshot folder. The tool will download [b]dxgi.dll[/b] from [b]reshade.me[/b] and update [b]ReShade.ini[/b].[/*]
[*]Launch through MO2 normally, or launch Skyrim from the game folder for a direct/manual install.[/*]
[/list]

Your Skyrim directory is the folder that contains [b]SkyrimSE.exe[/b].

[b]MO2 Root Builder:[/b] install the archive as a normal MO2 mod. Run [b]Wyrdlight ReShade Setup.exe[/b] from inside the installed mod's [b]root[/b] folder. If it detects your [b]Stock Game[/b] folder, accept it and choose to configure the mod folder for Root Builder. With LoreRim-style Root Builder setups, deployment happens when you launch through MO2 according to the modlist's Root Builder settings.

[b]Manual install with the setup tool:[/b] open the archive, copy the contents of the [b]root[/b] folder into your Skyrim directory or Stock Game folder, then run [b]Wyrdlight ReShade Setup.exe[/b] from there. Do not copy the [b]root[/b] folder itself.

[b]Full manual fallback:[/b] if you cannot run the setup tool, copy the contents of the [b]root[/b] folder into your Skyrim directory or Stock Game folder. Install the official ReShade full add-on build for [b]SkyrimSE.exe[/b] using the DirectX 10/11/12 option, or place the ReShade loader there as [b]dxgi.dll[/b]. Then edit [b]ReShade.ini[/b] so the preset, shader, and texture paths point to your game folder:

[list]
[*][b]CurrentPresetPath[/b] and [b]PresetFiles[/b] should point to [b]Wyrdlight.ini[/b].[/*]
[*][b]EffectSearchPaths[/b] should point to [b]reshade-shaders\Shaders[/b].[/*]
[*][b]TextureSearchPaths[/b] should point to [b]reshade-shaders\Textures[/b].[/*]
[*]If using ENB, leave proxy loading disabled unless you know you need it.[/*]
[/list]

[b]dxgi.dll:[/b] this is the ReShade loader. The setup tool downloads and installs it for you. It must end up beside [b]SkyrimSE.exe[/b], either directly or through Root Builder deployment.

For a normal Skyrim install, this is usually the expected Steam path:

[b]C:\Program Files (x86)\Steam\steamapps\common\Skyrim Special Edition[/b]

For a Wabbajack modlist that uses a [b]Stock Game[/b] folder, the [b]Stock Game[/b] folder is your Skyrim directory. Use the folder that contains the modlist's copy of [b]SkyrimSE.exe[/b].

Wabbajack example:

[b]G:\Modding\LoreRim\Stock Game[/b]

[heading][color=#8ECFFB]Expected File Layout[/color][/heading]
After setup and Root Builder deployment, or after a direct manual install, these files and folders should be in the same folder as [b]SkyrimSE.exe[/b]:

[list]
[*][b]dxgi.dll[/b][/*]
[*][b]ReShade.ini[/b][/*]
[*][b]Wyrdlight.ini[/b][/*]
[*][b]reshade-shaders\Shaders[/b][/*]
[*][b]reshade-shaders\Textures[/b][/*]
[/list]

If you use Mod Organizer 2 with Root Builder, keep the archive's [b]root[/b] folder structure. Root Builder should deploy those files to the Skyrim directory. For Wabbajack Stock Game setups, that means the [b]Stock Game[/b] folder. They should not end up inside [b]Data[/b].

[line]

[heading][color=#8ECFFB]Use[/color][/heading]
Launch Skyrim normally after running [b]Wyrdlight ReShade Setup.exe[/b]. If you use MO2 Root Builder, launch through MO2 as usual; Root Builder should deploy root files according to your modlist's settings.

Open the ReShade menu in game with [b]Ctrl+F12[/b] if you need to confirm that [b]Wyrdlight.ini[/b] is selected.

Toggle ReShade effects on or off with [b]Scroll Lock[/b].
